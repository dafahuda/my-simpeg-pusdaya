# Struktur Folder Dokumentasi

Dokumen ini merangkum struktur folder dokumentasi yang disepakati setelah migrasi.

## Struktur Target

```text
docs/
├── README.md
├── architecture/
│   └── folder-structure.md
└── schema/
    └── erd_schema_simpeg_bkn_supabase.md
```

## Pembagian Konten

- `architecture/` untuk dokumentasi arsitektur, keputusan struktur, dan panduan tata letak dokumen.
- `schema/` untuk dokumentasi skema data, termasuk ERD SIMPEG Supabase.
- `README.md` menjadi index utama agar navigasi antardokumen tetap jelas.
